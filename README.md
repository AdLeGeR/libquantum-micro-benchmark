компиляцию можно выполнить командой `./compile.sh` (предварительно сделайте файл исполняемым командой `chmod +x compile.sh`)\
для ветки time_logs:
 - `./compile.sh log.txt 50000` первый аргумент - имя файла(файл создаётся в папке ../logs), по умолчанию log.txt. второй аргумент - размер массива для записи логов, по умолчанию 50000.
 - выходной файл сохраняется в ../bin/time_logs\
bin_logs:
 - `./compile.sh \{1,2,123,1234\} 4 arglog` первый аргумент - массив номеров вызовов функции, аргументы которой нужно записать. Второй аргумент - длина массива. третий аргумент - имя файла для записи, по умолчанию arglog. Аргументы записываются в файл ../logs/arglog+номер_вызова
 - выходной файл сохраняется в ../bin/bin_logs\
run_hotspots:
 - `./compile.sh` без аргументов
 - файл сохраняется в ../bin/run_hotspots
 - запуск программы: `../bin/run_hotspots arglog1 100 arglog2 100`. аргументы задаются парами: путь_к_файлу количество повторений
